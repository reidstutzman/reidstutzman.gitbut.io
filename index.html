Lighting Assessment Tool Development

Designed and built an automated lighting assessment calculator to streamline analysis for energy audits. Consolidated existing spreadsheet tools into a user-friendly Excel interface, enabling interns to instantly generate lighting upgrade recommendations.

Key features of the tool include:

  • Automated report table population from input data to reduce duplicate manual entries.
  • Customizable for various lighting fixture configurations to handle diverse client needs.
  • Luminosity comparisons between current and proposed systems to ensure appropriate brightness.
  • Footcandle calculation integrating inverse square rule to quantify and address existing lighting deficiencies.

The tool has increased intern productivity by 25%, while producing more tailored, actionable lighting retrofit suggestions aligned to each facility's unique requirements.
